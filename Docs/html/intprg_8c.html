MTU4のTGRA割り込み、0.25msec毎に関数が実行される。

正常にマイコンが動作している場合、250usec周期で

距離センサ0 → 距離センサ1 → 距離センサ2 → ジャイロセンサ の順でスキャンする。

従って、1センサのスキャン周期は1msecとなる。
